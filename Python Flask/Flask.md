# Flask

使用python编写的轻量级Web应用框架

基于WSGI（Web Server Gateway Interface）和Jinja2模板引擎

微框架，使用简单核心，用扩展增加功能



## 特点

1. 轻量级、简洁
2. 灵活：提供基本框架，无强制项目布局与组件
3. 可扩展性
4. 内置开发服务器
5. RESTful支持



## 组成

1. Flask应用实例：核心，通过创建Flask对象初始化应用
2. 路由与视图函数：
   路由将URL映射到视图函数
   视图函数处理请求并返回响应
3. 模板系统：使用Jinja2模板引擎渲染HTML页面，将数据动态插入页面
4. 请求和响应：处理HTTP请求，生成响应，支持多种HTTP方法
